# Анализ конференций TED

**Задачи проекта:** Разработать аналитический дашборд на основе исторических данных конференций TED для поддержки принятия решений при организации нового мероприятия.

**Описание проекта:** Создан дашборд, который позволяет анализировать популярность тем выступлений и реакцию аудитории, определять среднее количество выступающих на конференции, оценивать оптимальную длительность выступлений для планирования временного графика, а также выявлять наиболее интересных для зрителей спикеров. Инструмент предоставляет визуализированные инсайты для организации мероприятия в духе классических конференций TED.

*Ссылка на дашборд:*

# Визуализация метрик сервиса доставки еды

**Задачи проекта:** Разработать интерактивный дашборд для комплексного анализа ключевых метрик сервиса доставки еды.

**Описание проекта:** Создан дашборд, позволяющий отслеживать ежедневную активность пользователей (DAU), конверсию в заказы, уровень удержания клиентов, динамику среднего чека, а также оценивать вклад популярных ресторанов и блюд в общую выручку. Инструмент предназначен для выявления трендов и поддержки принятия управленческих решений по оптимизации работы сервиса.

*Ссылка на дашборд:*
- https://datalens.yandex/jvra6s07nrbm5
