# Визуализация метрик сервиса доставки еды

**Задачи проекта:** Разработать интерактивный дашборд для комплексного анализа ключевых метрик сервиса доставки еды.

**Описание проекта:** Создан дашборд, позволяющий отслеживать ежедневную активность пользователей (DAU), конверсию в заказы, уровень удержания клиентов, динамику среднего чека, а также оценивать вклад популярных ресторанов и блюд в общую выручку. Инструмент предназначен для выявления трендов и поддержки принятия управленческих решений по оптимизации работы сервиса.
